# تطبيق Streamlit لتحليل الصور ومنشورات X

## الملفات
- app.py
- image_analysis_points.py
- requirements.txt
- packages.txt
- .streamlit/config.toml

## التشغيل محليًا
```bash
pip install -r requirements.txt
streamlit run app.py
```

## ملاحظة
ملف `image_analysis_points.py` ملف مساعد فقط، وملف التشغيل الرئيسي هو `app.py`.
